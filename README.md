# FitPlan 💪

FitPlan is a responsive workout and nutrition planner built with HTML, CSS, and JavaScript. It allows users to paste their entire weekly gym routine once and automatically organizes it into a structured workout schedule.

## Features

### Workout Planner

* Paste an entire weekly workout routine
* Automatically structures workouts by day
* Displays only today's workout on the dashboard
* Supports multiple exercises, sets, and reps
* Shows rest-day messages automatically

### Dashboard

* Dynamic greeting based on time of day
* Displays today's workout only
* Rest-day notifications for non-training days
* Clean and mobile-friendly interface

### Nutrition Calculator

* Daily calorie calculator
* Lean bulk, maintenance, and cutting targets
* Daily protein target calculation based on body weight

## Technologies Used

## Local Setup

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/fitplan.git
```

Open the project folder:

```bash
cd fitplan
```

Run locally using VS Code Live Server or simply open `index.html`.
